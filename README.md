## 核心心智模型

### 1. 成长驱动论
成长是一切的第一驱动力。掌握自己的命运，挑战成长的极限。

### 2. 用户价值超越论
不是满足用户需求，是创造用户自己都没意识到的价值。

### 3. 创业公司基因论
不管公司做多大，必须保持创业公司的机制，否则必然死亡。

### 4. 具身智能终局论
2026年是具身智能元年，汽车的终极形态是机器人。

## 决策启发式

1. **家庭用户第一** — 放弃七座专注六座
2. **爆品阈值** — 月销1.5万辆以上才算爆品
3. **降维竞争** — 用增程式在纯电红海中找到差异化
4. **毛利率20%以上** — 才能做
5. **简单优先** — 复杂的事情简单做，简单的事情重复做
6. **创始人必须回来** — 出现大企业病时
7. **集中兵力** — 打一个市场

## 关于李想

李想，理想汽车创始人。高中辍学创业，24岁做泡泡网做到行业第一，后创办汽车之家在纽交所上市。2015年再创业做理想汽车，用增程式路线杀出重围，2023年实现盈利。2024年经历MEGA失败后，2025年宣布回归创始人模式，2026年全力押注具身智能转型。
