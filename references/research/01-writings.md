### 2.1 《创业20年经历的五个楼层》

| 属性 | 内容 |
|------|------|
| **发布时间** | 2019年2月10日（春节假期最后一天） |
| **发布渠道** | 李想本人微博 |
| **一手性** | ★★★★★（李想亲笔） |
| **来源** | https://mp.weixin.qq.com/s/oNjJ-emZHomwJwZa0OB4NA |
| **可信度** | 极高。李想本人微博原文，钛媒体转载 |

**核心内容**：

五个楼层模型：
1. **一层楼**：生为一个普通的人（18岁前，随波逐流）
2. **二层楼**：成为一个优秀的人（高三创业，做个人网站）
3. **三层楼**：成为一个优秀的管理者（手把手带团队，2004-2008）
4. **四层楼**：成为一个优秀的领导者（关注人，培养管理者，2008-2018）
5. **五层楼**：成为一个顶尖的领袖（2018年后，智能电动车赛道）

**关键原话**：
> "人因为痛苦而改变，人因为受益而坚持。我们的真实的实力远比我们想象的更强，提高认知的楼层是我人生中最关键的收获。"

**附注**：本文同时澄清了"被赶出汽车之家"的传言：
> "2011年想离开汽车之家去创业，我和秦致说了想法。秦致说我们必须把汽车之家做到50亿美金以上才能让我走。我就留了下来，也一起做到了50亿美金以上的市值，在2015年6月离开了汽车之家。"

### 2.2 2025年Q3财报电话会发言

| 属性 | 内容 |
|------|------|
| **发布时间** | 2025年11月27日 |
| **发布渠道** | 财报电话会（实录） |

| **来源** | https://k.sina.cn/article_7879848900_1d5acf3c401902ttic.html |
| **可信度** | 极高。一手发言记录 |

**核心内容——三个关键选择**：

#### 选择一：组织——回归创业公司管理模式
> "过去三年，我和创业团队努力学习职业经理人的管理体系，逼迫自己接受各种变化，但是我们却变成了越来越差的自己。而英伟达和特斯拉今天仍以创业公司的方式管理。如果全世界最强的那些公司都可以使用创业公司的管理方式，我们有什么理由放弃自己最擅长的？"

**创业公司管理模式核心四件事**：
1. 更多的对话，深度的对话，而非更多的汇报
2. 聚焦用户价值，而非只是交付
3. 识别关键问题，而不是创造信息不对称

#### 选择二：产品——具身智能机器人
> "电动车的选择不是不好，而是不够。智能终端的选择也不是不好，而是不够。只有选择具身智能这个最难的题，我们才可以真正改变用户的生活，提供自动和主动服务的具身智能机器人产品。"

#### 选择三：技术——完整AI系统的全栈自研
> "要想做好具身智能，最重要的是构建一套不同于语言智能的AI系统"

## 十、关键发现

1. **系统性长文较少**：李想的方法论主要通过课程（得到16讲）、内部培训（15期）、重要节点发言（全员会、财报会）传播，而非传统博客/文章。

2. **核心信念高度一致**：从2019年"五个楼层"到2026年"具身智能"，"成长驱动"、"用户价值"、"创业公司模式"等核心概念一以贯之。

3. **术语迭代轨迹**：
   - 2015-2023：**"移动的家"**为核心定位
   - 2024-2025：**"硅基家人"**开始出现
   - 2026：**"具身智能"**成为战略核心词

4. **公开资料丰富**：虽然李想本人著书较少，但演讲、采访、内部发言记录较为完整，为理解其思维框架提供充足素材。
